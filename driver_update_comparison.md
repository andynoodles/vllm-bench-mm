# 驅動程式更新效能比較 (NVIDIA 595 vs 580)

比較 NVIDIA 595 驅動程式與原始 580 驅動程式的效能差異。
*(註：吞吐量越高越好。執行時間和延遲越低越好。正百分比代表相對於原始 580 驅動的效能提升。)*

### 併發數 16 (`officail_vllm_FP8c16.txt`)

| 測試指標 | 580 驅動程式 | 595 驅動程式 | 提升比例 |
| :--- | :--- | :--- | :--- |
| **測試時間 (s)** | 130.98 | 127.89 | **+ 2.36%** |
| **總 Token 吞吐量 (tok/s)** | 806.29 | 825.73 | **+ 2.41%** |
| **輸出 Token 吞吐量 (tok/s)** | 58.64 | 60.05 | **+ 2.40%** |
| **平均首字延遲 TTFT (ms)** | 15462.13 | 14731.96 | **+ 4.72%** |
| **平均每字元延遲 TPOT (ms)** | 204.53 | 201.18 | **+ 1.64%** |
| **平均字元間延遲 ITL (ms)** | 203.86 | 200.40 | **+ 1.70%** |
