# BIOS 更新效能比較

比較 BIOS 更新前後的效能差異。
*(註：吞吐量越高越好。執行時間和延遲越低越好。正百分比代表效能提升。)*

### 併發數 16 (`officail_vllm_FP8c16.txt`)

| 測試指標 | 580 驅動 (原始) | 580 驅動 (BIOS 更新) | 提升比例 |
| :--- | :--- | :--- | :--- |
| **測試時間 (s)** | 130.98 | 129.50 | **+ 1.13%** |
| **總 Token 吞吐量 (tok/s)** | 806.29 | 815.45 | **+ 1.14%** |
| **輸出 Token 吞吐量 (tok/s)** | 58.64 | 59.30 | **+ 1.13%** |
| **平均首字延遲 TTFT (ms)** | 15462.13 | 16337.48 | **- 5.66%** *(退步)* |
| **平均每字元延遲 TPOT (ms)** | 204.53 | 196.75 | **+ 3.80%** |
| **平均字元間延遲 ITL (ms)** | 203.86 | 196.14 | **+ 3.79%** |
